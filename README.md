# Image Classification using CNN – Fashion-MNIST

## Project Overview
This project implements an image classification system using a Convolutional
Neural Network (CNN) trained on the Fashion-MNIST dataset.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## Project Files
- `fashion_mnist_cnn.ipynb` – Complete implementation
- `screenshots/` – Training, evaluation, and prediction outputs
- `presentation.pptx` – Project presentation
